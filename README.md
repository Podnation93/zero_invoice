# zero_invoice

A local-first invoicing, expense, payroll, and tax reporting application.

## Overview

This project is a self-hosted web-based application that replicates the core functionalities of Xero. It is built with a Node.js backend, a React frontend, and uses SQLite for local data storage.

## Current Status

Project initialized. Backend setup with Node.js, Express, and SQLite.
